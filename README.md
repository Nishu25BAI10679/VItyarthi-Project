# VItyarthi-Project
Project Title: Budget Manager

Overview:
The Budget Manager programme is designed to help user in managing and monitoring their budget and expenses.It keep the user updated by giving regular alerts on low budget and high expenses.It analyse their daily spending pattern to prepare a proper monthly expense chart.

Feature:
1.Set monthly budget
2.Add expenses and categorize them
3.View list of expenses
4.Check remaining budget
5.Receives warnings and alerts when funds run low
6.Persistent data storage in a simple text file

Technologies & Tools:
1.Python: Main programming language
2.Built-in modules: os for file handling, datetime for date calculations
3.Text file storage: Minimal dependencies, portable data persistence
4.Command-line interface: For Easy cross-platform interaction

Installation & Running:
1.Make sure Python is installed on your system
2.Download the project repository
3.Navigate to the project folder via terminal/command prompt
4.Run the program by typing:python budget_manager.py
5.Follow on-screen prompts to set or load your budget and manage expenses.

TestingInstructions:
1.Test the program by running the script and exercising all menu options:
 a.Input valid and invalid budget to test validation
 b.Add positive and invalid expense amounts to confirm input checking
 c.View expenses and confirm that they display correctly
 d.Check remaining funds and different alert warnings
2.For automated tests, Python’s unittest framework can be used to validate functions like data loading/saving and calculations (test scripts can be created separately).
